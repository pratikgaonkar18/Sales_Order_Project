# Order Workflow Frontend

React application for the internal Sales Order workflow prototype.

## Pages

- Dashboard page
	- Active order grid
	- Search by customer, part number, sales order, reference serial
- Create Order page
	- Header fields and first line item
	- Submits to backend order API

## Run locally

1. Start backend first at `http://localhost:8080`.
2. In this folder run:

```powershell
npm run dev
```

Vite serves the UI at `http://localhost:5173`.

## Build

```powershell
npm run build
```
